Android 16KB Page Size Checker

Công cụ kiểm tra tương thích 16KB memory page size cho APK/AAB Android. Google Play yêu cầu tất cả native library (.so) phải hỗ trợ 16KB alignment từ năm 2025.

📌Tại sao cần kiểm tra?

Android 15 trở lên hỗ trợ 16KB page size để cải thiện hiệu năng. Google Play bắt buộc tất cả app có native library phải tương thích kể từ 1/11/2025 cho app mới và 1/6/2026 cho app update.

🎮Dành cho Unity Game Dev

Unity IL2CPP build thường tạo ra libil2cpp.so và libunity.so với 4KB alignment. Tool này giúp bạn phát hiện và sửa lỗi trước khi upload Google Play.

🔧Hướng dẫn sửa lỗi 16KB Page Size

Thêm flag sau vào linker để rebuild với 16KB alignment:

🔨 NDK / Clang (khuyến nghị)
-Wl,-z,max-page-size=16384
📄 Android.mk
LOCAL_LDFLAGS += -Wl,-z,max-page-size=16384
🏗️ CMakeLists.txt
target_link_options(${TARGET} PRIVATE -Wl,-z,max-page-size=16384)
🎮 Unity IL2CPP (Project Settings → Player → Android)
Additional IL2CPP Arguments: -Wl,-z,max-page-size=16384
📦 app/build.gradle
android { defaultConfig { externalNativeBuild { cmake { arguments "-DANDROID_LDFLAGS=-Wl,-z,max-page-size=16384" } } } }

📖Tổng quan về 16KB Page Size

Android 15 (API 35) giới thiệu hỗ trợ 16KB memory page size. Các thiết bị dùng chip ARM mới (Cortex-X) có thể hoạt động ở chế độ 16KB page để:

  • Giảm TLB miss → tăng hiệu năng CPU
  • Cải thiện thời gian khởi động app
  • Tối ưu cache hiệu quả hơn

Nếu native library (.so) không được compile với 16KB alignment, app sẽ bị từ chối trên Google Play hoặc crash trên thiết bị 16KB.

🔍Cách kiểm tra

Dùng readelf để kiểm tra LOAD segment alignment:

readelf -l libfoo.so | grep LOAD

Nếu cột Align là 0x4000 (16384) → ✅ OK
Nếu là 0x1000 (4096) → ❌ Cần rebuild

🎮Unity IL2CPP — Hướng dẫn sửa
1️⃣ Unity 2022.3 LTS / 2023 / 6000

Project Settings → Player → Android → Publishing Settings → Additional IL2CPP arguments:

-Wl,-z,max-page-size=16384
2️⃣ Qua baseProjectTemplate.gradle
android { defaultConfig { externalNativeBuild { cmake { arguments "-DANDROID_LDFLAGS=-Wl,-z,max-page-size=16384" } } } }
3️⃣ Kiểm tra sau khi build
python3 apk16kb_checker.py MyGame.apk
